# Frozen Icon 
A visual indicator next to your HP bar when frozen or entangled.

This plugin detects when your player is affected by freezing or binding spells (e.g., Ice Barrage, Entangle) and displays an icon beside the HP bar indicating the duration of the effect. Useful for PvP and PvM awareness.

## Features

- **Freeze / Bind Indicator**  
  Displays an icon next to the HP bar when your character is frozen or entangled (e.g., Ice Barrage, Entangle).

- **Immunity Icon**  
  Optional **shield icon** appears when your character has freeze immunity after being unfrozen.

- **Freeze Timer**  
  Shows a countdown timer indicating how long the freeze or bind effect will last.

- **Teleblock Indicator**  
  Detects when your character is teleblocked and displays an indicator for quick awareness.

- **Configurable Options**  
  All indicators (freeze, immunity, teleblock, timer) can be toggled on or off in the plugin settings.
